# 快速开始

本文档将帮助你在 5 分钟内快速部署 NexusAuth 统一身份认证平台。

## 前置要求

在开始之前，请确保你已具备以下环境：

- .NET 9.0 SDK
- Node.js 20+
- PostgreSQL 16+

## 快速部署步骤

### 步骤 1：克隆项目

```bash
git clone https://github.com/your-repo/NexusAuth.git
cd NexusAuth
```

### 步骤 2：启动 PostgreSQL（Docker）

```bash
docker run -d \
  --name nexusauth-db \
  -e POSTGRES_PASSWORD=your-password \
  -e POSTGRES_USER=nexusauth \
  -e POSTGRES_DB=nexusauth \
  -p 5432:5432 \
  postgres:16
```

### 步骤 3：配置数据库连接

修改 `src/NexusAuth.Host/appsettings.json`：

```json
{
  "ConnectionStrings": {
    "DefaultConnection": "Host=localhost;Port=5432;Database=nexusauth;Username=nexusauth;Password=your-password"
  }
}
```

### 步骤 4：运行数据库迁移

```bash
cd src/NexusAuth.Host
dotnet ef database update
```

### 步骤 5：启动认证服务

```bash
dotnet run
```

服务启动后访问 http://localhost:5100

### 步骤 6：配置 OAuth 客户端

执行 seed.sql 添加基础数据：

```bash
psql -U nexusauth -d nexusauth -f NexusAuth.Workbench/NexusAuth.Workbench.Api/seed.sql
```

### 步骤 7：启动 Workbench API

```bash
cd NexusAuth.Workbench
dotnet run --project NexusAuth.Workbench.Api
```

### 步骤 8：启动 Dashboard

```bash
cd NexusAuth.Workbench/NexusAuth.Workbench.Dashboard
npm install
npm run dev
```

访问 http://localhost:5273 使用统一账号登录。
