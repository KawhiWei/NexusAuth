# 环境准备

本文档详细介绍 NexusAuth 开发环境的所有要求。

## 软件要求

### 必需软件

| 软件 | 版本 | 说明 |
|------|------|------|
| .NET SDK | 9.0+ | 后端运行时 |
| Node.js | 20.x+ | 前端包管理 |
| PostgreSQL | 16.x | 关系型数据库 |
| Git | 2.x | 版本控制 |

### 可选软件

| 软件 | 版本 | 说明 |
|------|------|------|
| Docker | Latest | 容器化部署 |
| Visual Studio Code | Latest | 代码编辑器 |
| JetBrains Rider | 2024.2+ | .NET IDE |
| DBeaver | Latest | 数据库管理 |

## 环境验证

### 验证 .NET

```bash
dotnet --version
# 输出: 9.0.x
```

### 验证 Node.js

```bash
node --version
# 输出: 20.x.x
npm --version
# 输出: 10.x.x
```

### 验证 PostgreSQL

```bash
psql --version
# 输出: psql (PostgreSQL) 16.x
```

## 项目结构

```
NexusAuth/
├── src/
│   ├── NexusAuth.Host/           # OAuth Provider (端口: 5100)
│   ├── NexusAuth.Workbench.Api/  # Workbench API (端口: 5051)
│   ├── NexusAuth.Workbench.Dashboard/ # Dashboard (端口: 5273)
│   ├── NexusAuth.Extension/     # 扩展类库
│   ├── NexusAuth.Domain/       # 领域模型
│   ├── NexusAuth.Persistence/  # 数据层
│   └── NexusAuth.Application/   # 应用服务
├── document/                    # 文档
│   └── diagrams/               # 架构图
└── demo/                       # 示例应用
```

## 各服务端口配置

| 服务 | 默认端口 | 配置文件 |
|------|---------|----------|
| NexusAuth.Host | 5100 | appsettings.json |
| NexusAuth.Workbench.Api | 5051 | appsettings.json |
| NexusAuth.Workbench.Dashboard | 5273 | vite.config.ts |

## 开发工具推荐

### VS Code 插件

- C# (Microsoft)
- ESLint
- Prettier
- GitLens

### 数据库工具

推荐使用 DBeaver 或 pgAdmin 连接数据库：

```bash
# Docker 启动 pgAdmin
docker run -d \
  --name pgadmin \
  -e PGADMIN_DEFAULT_EMAIL=admin@example.com \
  -e PGADMIN_DEFAULT_PASSWORD=admin \
  -p 8080:80 \
  dpage/pgadmin4
```
