# 高级配置

本文档介绍 NexusAuth 的高级功能和配置选项。

## Token 配置

### Access Token 有效期

在 `TokenService` 中配置：

```csharp
// 默认 1 小时
var expiresIn = 3600;
```

### 刷新 Token

启用刷新 Token：

```sql
-- 在 oauth_clients 表中配置
UPDATE oauth_clients 
SET allowed_grant_types = '["authorization_code", "refresh_token"]'
WHERE client_id = 'your-client-id';
```

### CORS 配置

```csharp
services.AddCors(options =>
{
    options.AddPolicy("AllowFrontend", policy =>
    {
        policy.WithOrigins("http://localhost:5273")
            .AllowAnyHeader()
            .AllowAnyMethod()
            .AllowCredentials();
    });
});
```

## 自定义 FlowState 存储

### 使用 Redis

```csharp
public class RedisFlowStateStore : IFlowStateStore
{
    private readonly IConnectionMultiplexer _redis;

    public async Task AddAsync(string state, FlowState flowState, CancellationToken ct)
    {
        var key = $"flow:{state}";
        var value = JsonSerializer.Serialize(flowState);
        await _redis.GetDatabase().StringSetAsync(key, value, TimeSpan.FromMinutes(10));
    }

    public async Task<FlowState?> GetAsync(string state, CancellationToken ct)
    {
        var key = $"flow:{state}";
        var value = await _redis.GetDatabase().StringGetAsync(key);
        return value.HasValue 
            ? JsonSerializer.Deserialize<FlowState>(value!) 
            : null;
    }

    public async Task RemoveAsync(string state, CancellationToken ct)
    {
        var key = $"flow:{state}";
        await _redis.GetDatabase().KeyDeleteAsync(key);
    }
}
```

### 注册 Redis Store

```csharp
services.AddSingleton<IFlowStateStore, RedisFlowStateStore>();
```
