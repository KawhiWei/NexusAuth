# 常见问题

本文档收集了 NexusAuth 使用过程中的常见问题及解决方案。

## 认证相关

### Q: 登录后刷新页面又跳转回登录页

**A:** 检查以下几点：
1. 确认登录后 Cookie 是否正确写入
2. 检查 /auth/me 接口是否返回正确
3. 检查前端 cachedAuthStatus 是否正确缓存

### Q: redirect_uri 不匹配

**A:** 
```
错误: invalid_request_redirect_uri
```
确保 OAuth 客户端配置的 redirect_uri 与应用完全一致，包括：
- 协议（http/https）
- 端口号
- 路径（包括尾斜杠）

### Q: PKCE 验证失败

**A:**
```
error: invalid_code_verifier
```
1. 确保 code_challenge_method 设置为 S256
2. 确保 code_verifier 长度在 43-128 字符之间
3. 确保 code_verifier 只包含 URL 安全字符

### Q: invalid_client

**A:** 检查：
1. client_id 是否正确
2. client_secret 是否正确（如果是 client_secret_basic）
3. 客户端是否启用（is_active = true）

## 前端相关

### Q: 登录页死循环刷新

**A:** 检查：
1. axios 401 拦截器不应该拦截 /auth/ 相关请求
2. 检查登录页是否有 try-catch 处理错误

```typescript
// 修复示例
if (error.response?.status === 401 && 
    !error.config?.url?.includes('/auth/')) {
  window.location.href = '/login';
}
```

### Q: TypeScript 类型错误

**A:** 
```
error TS2304: Cannot find name 'xxx'
```
运行构建检查：
```bash
npm run typecheck
```

## 后端相关

### Q: 登出后还能访问接口

**A:** 
1. 确保 Cookie 正确清除
2. 检查登出接口是否有权限验证

### Q: token 刷新失败

**A:** 
1. 确保客户端配置了 refresh_token grant
2. 检查 refresh_token 是否过期

### Q: CORS 错误

**A:** 
```
Access to XMLHttpRequest has been blocked by CORS policy
```
配置 CORS：
```csharp
services.AddCors(options =>
{
    options.AddPolicy("AllowFrontend", policy =>
    {
        policy.WithOrigins("http://localhost:5273")
            .AllowAnyHeader()
            .AllowAnyMethod()
            .AllowCredentials();
    });
});
```

## 数据库相关

### Q: 数据库连接失败

**A:** 
```
error: connection to server at "localhost" failed
```
1. 检查 PostgreSQL 是否运行
2. 检查连接字符串
3. 检查端口 5432 是否开放

### Q: 迁移失败

**A:** 
```
error: relation "table" already exists
```
删除现有数据库重建：
```bash
dotnet ef database drop
dotnet ef migrations remove
dotnet ef migrations add InitialCreate
dotnet ef database update
```

## 部署相关

### Q: Docker 容器无法启动

**A:** 
1. 检查端口是否冲突
2. 检查环境变量
3. 查看容器日志：```bash
docker logs <container-id>
```

### Q: HTTPS 配置

**A:** 
配置 Kestrel：
```json
{
  "Kestrel": {
    "Endpoints": {
      "Https": {
        "Url": "https://0.0.0.0:5001",
        "Certificate": {
          "Path": "certificate.pfx",
          "Password": "password"
        }
      }
    }
  }
}
```

## 性能相关

### Q: Token 生成慢

**A:** 
检查是否有同步操作，优化为异步：
```csharp
// 避免
var key = RSA.Create();

// 推荐
await Task.Run(() => RSA.Create());
```
